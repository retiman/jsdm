JSDM::NaturalLoops

This class calculates the natural loops in a directed graph, given the back edges.

Implements the algorithm found on page 192 of Advanced Compiler Design and Implementation by Steven S. Muchnick.

Public Instance Methods

find(graph, back_edges) click to toggle source

Returns a set of “loops” (which are in turn a set of nodes), given a DirectedGraph and the back edges found from a DepthFirstSearch, if any exist.
