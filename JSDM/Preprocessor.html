process_single(source) click to toggle source

Processes a single source file for its dependencies so that they may be resolved. Here’s how this function does it:

  • Open the source file and read it, taking all the lines that match the comment_pattern

  • Select the lines that match the require_pattern from those lines

  • Split each require statement with a comma in it

  • Flatten the array of dependencies

  • Strip out the trailing whitespace
